APIs and Frameworks Used

  • Vue
    • Helped to build the UI and functions needed for the UI
  • Leaflet
    • API used to create the map and data included in the map(markers and icons)
    • Used a leaflet geocode control for the address and latitude and longitude search function
  • Nominatim
    • Used to search for addresses
    • Converts latitude and logitude coordinates to addresses
  • St. Paul Crime
    • API created to access a St. Paul Crime database
    • Can access and filter incidents, neighborhoods, and types of crimes
  • Bootstrap
    • Used to format the website and components in the website
  • JQuery
    • Used to get data from the database
